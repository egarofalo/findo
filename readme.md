# API actions

## Get movies
- ENDPOINT: <code>/movies</code>
- METHOD: <code>GET</code>

## Get movie
- ENDPOINT: <code>/movies/{movie_id}</code>
- METHOD: <code>GET</code>

## Create movie
- ENDPOINT: <code>/movies</code>
- METHOD: <code>POST</code>

## Update movie
- ENDPOINT: <code>/movies/{movie_id}</code>
- METHOD: <code>PUT</code>

## Delete movie
- ENDPOINT: <code>/movies/{movie_id}</code>
- METHOD: <code>DELETE</code>
